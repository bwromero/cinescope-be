================================================================================
                         CINESCOPE MVP - PROJECT SPECIFICATION
================================================================================

CORE PURPOSE
------------
A movie discovery app where users can browse, search, and save movies to a 
personal watchlist.


================================================================================
                              USER STORIES
================================================================================

#   FEATURE              DESCRIPTION
------------------------------------------------------------------------------
1   Browse Trending      See what's popular this week
2   Browse by Genre      Filter movies by Action, Comedy, Horror, etc.
3   Search Movies        Find any movie by title
4   View Details         See synopsis, rating, cast, trailer, similar movies
5   Save to Watchlist    Add/remove movies from personal list
6   View Watchlist       See all saved movies

================================================================================
                         BACKEND ENDPOINTS (Spring Boot)
================================================================================

MOVIES (from TMDB)
------------------
GET  /api/movies/trending              → Trending this week
GET  /api/movies/popular?page=1        → Popular movies (paginated)
GET  /api/movies/search?query=batman   → Search by title
GET  /api/movies/{id}                  → Movie details
GET  /api/genres                       → All genres

DISCOVER
--------
GET  /api/movies/discover?genre=28&year=2024&page=1

WATCHLIST (uses database)
-------------------------
GET     /api/watchlist                 → Get user's watchlist
POST    /api/watchlist                 → Add movie to watchlist
DELETE  /api/watchlist/{movieId}       → Remove from watchlist
